# Surge 配置文件与模块集合

这是一个用于存放 Surge (iOS/macOS) 网络代理工具的配置、模块及脚本的仓库。

## 📁 目录说明

* **/Profiles**: 存放完整的 `.conf` 配置文件。
* **/Modules**: 各种功能增强模块 (`.sgmodule`)。
* **/Scripts**: 配置文件中引用的 JavaScript 脚本。

## 🛠 模块介绍

| 模块名称 | 功能描述 |
| :--- | :--- |
| **广告拦截** | 过滤常见 App 开屏广告及网页弹窗。 |
| **功能增强** | 包含流媒体解锁、系统功能优化等。 |
| **自动签到** | 每日自动完成指定 App 的积分签到。 |

## 🚀 使用说明

### 1. 引用模块
在 Surge 管理界面中，选择「模块」->「安装新模块」，输入本项目中模块的原始链接（Raw URL）。

### 2. 使用配置
下载 `/Profiles` 目录下的配置文件，修改其中的 `Proxy` 节点信息后，导入 Surge 即可使用。
